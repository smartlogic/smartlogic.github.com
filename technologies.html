Going off the Rails into Elixir

July 25, 2019 — The web is not that young anymore; we’ve been writing Ruby on Rails applications for over a decade at this point. While Rails is still a great framework and we still write in it all the time, over the past few years we’ve become increasingly convinced that there’s a new and better way — and that language is Elixir.

Elixir/Phoenix

Elixir and Phoenix are a great choice for web applications that need to scale up. Examples of some high profile web applications built in Elixir and Erlang include WhatsApp, BleacherReport, Pinterest, AdRoll, Grindr, and Bet365.

React Native

We’ve been building in React Native since 2016, and it continues to be our favorite framework for mobile app development. In the recent past we’ve used React Native to deliver apps for the Baltimore Water Taxi, SpotCrime, and the National Committee for Quality Assurance.
